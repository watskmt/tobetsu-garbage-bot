# 当別町ごみ収集日Bot — 使い方ガイド

当別町のごみ収集日をLINEで簡単に確認できるBotです。

---

## 友だち追加

LINEで **「当別町ごみ収集日Bot」** を友だち追加してください。

---

## はじめに：地区の設定

最初に自分の地区を設定します。以下のいずれかを送信してください。

| 送信するメッセージ | 設定される地区 |
|-------------------|---------------|
| `1地区` | 弥生・園生・青山・弁華別など |
| `2地区` | 金沢・中小屋・東裏・蕨岱町 |
| `3地区` | 白樺町・下川町・末広・スウェーデンヒルズなど |
| `4地区` | 春日町・樺岱町・太美・高岡など |

設定後は「〇〇地区に設定しました」と返信されます。

---

## 収集日を確認する

地区設定後、以下のキーワードを送信してください。

| キーワード | 確認できる内容 |
|-----------|-------------|
| `今日` | 今日の収集 |
| `明日` | 明日の収集 |
| `１週間` | 今日から7日間の収集予定 |
| `今月` | 今日から30日間の収集予定 |

画面下部のクイックリプライボタンからも選べます。

---

## 返信の例

```
5月29日(金): 燃やせないごみ
6月2日(火): 燃やせるごみ・燃えないごみ
6月6日(金): 燃やせないごみ
```

収集のない日は表示されません。

---

## 毎日通知を受け取る

設定した時刻に、その日のごみ収集情報を自動でお知らせします。
収集のない日は通知を送りません。

### 通知の設定方法

| 送信するメッセージ | 動作 |
|-----------------|------|
| `通知設定` | 時刻選択ボタンを表示 |
| `通知7時` / `7時通知` / `毎朝8時` など | その時刻に毎日通知を設定（正時のみ） |
| `通知確認` | 現在の通知時刻を確認 |
| `通知オフ` | 通知を停止 |

### 例

```
「通知7時」と送信
→ 毎日 07:00 にごみ収集情報を通知します。
  停止するには「通知オフ」と送ってください。
```

## ごみの種別

| 種別 | 説明 |
|------|------|
| 燃やせるごみ | 可燃ごみ |
| 燃やせないごみ | 不燃ごみ（ピンク袋） |
| 燃えないごみ | 不燃ごみ（水色袋） |
| 資源物 | 缶・瓶・ペットボトルなど |
| スプレー缶 | スプレー缶・カセットボンベ |

## よくある質問

**Q. 地区がわからない**
A. 当別町のホームページまたは回覧板でご確認ください。

**Q. 収集日が実際と異なる**
A. 祝日・年末年始などの例外日は管理者が随時更新しています。最新情報は町の公式カレンダーもあわせてご確認ください。

**Q. 通知が来ない**
A. 「通知確認」と送信して設定を確認してください。収集のない日は通知を送信しません。

**Q. 返信が来ない**
A. しばらく時間をおいて再送信してください。改善しない場合は管理者にご連絡ください。
